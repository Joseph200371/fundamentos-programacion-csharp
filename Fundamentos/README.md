# Fundamentos de Programación

Ejercicios orientados a la práctica de conceptos básicos de programación en C#, enfocados en la lógica, el uso correcto de tipos de datos y la resolución de problemas simples.

## Contenidos
- Lectura de datos por consola
- Operaciones aritméticas
- Uso de variables y tipos de datos (`int`, `double`, etc.)
- Cálculo de superficies y porcentajes
- Conversión de unidades
- Promedios
- Descomposición matemática
- Aplicación de fórmulas sin valores fijos

## Organización
Cada carpeta corresponde a un **ejercicio independiente**, implementado como un **proyecto de consola**.

## Objetivo
Consolidar las bases de la programación estructurada y desarrollar un razonamiento lógico aplicable a problemas reales.
